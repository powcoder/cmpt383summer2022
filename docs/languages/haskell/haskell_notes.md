# Haskell Notes

## Installation

Please use [GHC](https://www.haskell.org/). You can install Haskell on Ubuntu
Linux with this command:

```bash
$ sudo apt install haskell-platform 
```

We'll mainly be using `ghci`, a Haskell interpreter.
